How To Use This Site

1. Start with the short summary.

Each project card explains what the experiment is trying to learn and why that matters in plain language.

2. Watch a clip or open a demo.

Where media exists, you can preview it immediately. Interactive viewers are linked directly from each card.

3. Go deeper only if you want.

Every section also links back to the GitHub source and project README for the full technical detail.

Board Structure Optimizer

AI-guided experiments that try to redesign the inside of a foil board so more of the rider’s force goes efficiently into the mast and foil.

This is the most mature structural experiment in the repo. It takes the TT60 board shape, applies rider and mast loads, and tries to discover internal rib and lattice structures that are lighter than a conventional build but still stiff where they need to be.

Proof Of Concept

Tuttle vs Dual-Track Insert Comparison

A side-by-side structural comparison that isolates only the mast insert architecture inside the same TT60 board.

This viewer holds the board outer shape fixed and lets you change only the insert dimensions and footprint shape. The goal is to compare how a Tuttle box and a dual-track cassette spread load into the shell and foam, without muddying the picture with mast flex or a different board shape.

RL Pump Foil Simulation

A reinforcement learning agent that learns to pump a foil and hold flight instead of dropping back into the water.

The repo includes a working custom-physics RL setup and an exploratory MuJoCo branch. The practical result is simple: an agent can learn a pumping rhythm that looks close to real rider timing and sustain flight for meaningful durations.

Foil Motion Capture

Pose extraction from foil videos so you can inspect body timing, rhythm, and movement patterns.

Feed the project a foil clip and it returns skeleton overlays, trace views, and biomechanical features. This makes it easier to compare different riders or techniques without needing expensive motion-capture hardware.

Interactive Hydrofoil Explainer

A visual teaching project aimed at making trim, pumping, balance, and foil behavior understandable to non-engineers.

Think of this as the long-term “show, don’t just tell” project for hydrofoiling. It is meant to explain why setups feel fast, firm, mushy, stable, or turny, using interactive scenes instead of forum folklore.

Foil Design Optimization

A future experiment for optimizing front wings and stabilizers rather than just the board.

This project aims to use fast aerodynamic models to search for better combinations of foil profile, planform, and stabilizer interaction. It is still at the planning stage, but the intended output is a more systematic way to reason about foil speed, low-end lift, and stability.

Board Design Assistant

A planned conversational CAD workflow for changing foil-board geometry with natural language.

The long-term goal here is simple: describe a board change in plain language, see the geometry update, then feed that geometry straight into the structural and simulation tools. It is a planning-stage bridge between shaping, analysis, and experimentation.
